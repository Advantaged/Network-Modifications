# Network-Modifications
Modify &amp; Fix your Network under Linux

#### Scope of project/work:
Surprised by the lack of coherent instructions for the settings & changes of my own network (LAN) & as a newcomer to the subject... I am collecting my documentation & scripts here to get a clear professional fast & simple LAN configuration that can also help users & experts.
This includes the assignment of static IP addresses in modems/routers as well in admin- as in client-PCs, the creation of a perfectly functioning network bridge for the Virtual Machine Manager abbr. VMM (QEMU/KVM) that relieves CPU & standard network, the settings for Fritz.Box, TP-Link modem/router & MikroTik's switches.

- [**Assign-Static-IP**](https://github.com/Advantaged/Network-Modifications/tree/main/Assign-Static-IP):
 - This part of the project contain the assignment of a static IP on PC-side & the assignment of a NIC-port for the traffic of `macvtap`-bridge in VMM.
